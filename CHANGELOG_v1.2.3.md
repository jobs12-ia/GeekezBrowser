## 📝 技术说明

### 为什么不随机化字体？
- 字体 API 修改容易被检测
- 字体指纹权重很低（2%）
- Canvas/WebGL 已足够区分
- 商业工具也不随机化字体
- 风险大于收益

### TLS 指纹说明
- GeekEZ 使用真实 Chrome 浏览器
- TLS 握手由 Chrome 原生完成
- 与 User-Agent 完全匹配
- 与紫鸟等商业工具的 TLS 指纹相同
- 无法被检测为异常

### 性能优化原理
- 减少不必要的等待时间
- 限制缓存大小避免内存膨胀
- 自动清理临时文件
- 优化启动参数减少资源占用
